# Using the VS Code debugger in this project

## Debugging
- On 'Run and Debug' tab, select the option "Debug R-Package".
- Open the R script you want to debug and set breakpoints where needed.
- Press F5 to start debugging. Wait for the R session to start and the package to load.
- At the Debug Console prompt, source the R script you have opened for debugging.
- R will run the script and pause at the first breakpoint it encounters. You can inspect variables, step through code, and evaluate expressions in the Debug Console.
